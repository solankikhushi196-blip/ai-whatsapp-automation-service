# AI WhatsApp Automation for Service Businesses

## Overview
This project demonstrates how an AI-powered WhatsApp automation system
can handle customer support tasks for service-based businesses
such as cafes, salons, clinics, and local stores.

The system is designed as an AI Agent, not just a chatbot.

## Key Features
- Automated customer support on WhatsApp
- Working hours validation
- Service availability checks
- Order / booking handling
- Friendly, human-like responses
- Soft upselling

## Business Rules
- Business hours are fixed
- Service area is limited
- Specific actions require user confirmation
- Polite and local-language-friendly tone

## Use Cases
- Order status handling
- New service booking
- General customer queries
- Feedback collection

## Why This Matters
This project shows system-thinking, automation logic,
and AI agent design skills required for real-world businesses.
